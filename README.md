# nestup

![npm](https://img.shields.io/npm/v/nestup?style=flat-square)
![npm](https://img.shields.io/npm/dt/nestup?style=flat-square)

Generate a Nest project with:

- Docker support
- TypeORM
- Nodemon Setup
- Integration Tests
- Travis CLI
- MySQL/MariaDB
- TS Configs

## Usage

```sh
npx nestup my-new-project
```
